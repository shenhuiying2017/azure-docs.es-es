# <a name="azure-web-apps-documentation"></a>Documentación de Azure Web Apps

App Service Web Apps es una plataforma de procesos completamente administrada optimizada para hospedar sitios y aplicaciones web. Esta oferta de plataforma como servicio (PaaS) de Microsoft Azure le permite centrarse en su lógica de negocios, mientras que Azure se encarga de la infraestructura para ejecutar y escalar las aplicaciones.

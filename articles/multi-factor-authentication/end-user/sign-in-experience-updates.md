# <a name="updates-to-the-sign-in-experience-for-online-services"></a>Actualizaciones de la experiencia de inicio de sesión para los servicios en línea

Vamos a actualizar nuestra experiencia de inicio de sesión de los servicios en línea de Microsoft, como Office 365 y Azure. Aunque se trata solo de un cambio visual, observará que las pantallas parecen menos desordenadas y más sencillas, lo que le permitirá centrarse más fácilmente en lo que necesita hacer, es decir, iniciar sesión.

## <a name="current-ui-versus-new-ui"></a>Interfaz de usuario actual frente a la nueva

Consulte la interfaz de usuario existente en comparación con la nueva. La nueva interfaz de usuario es más sencilla y específica, con una apariencia más actualizada.

![Experiencia de inicio de sesión anterior de Microsoft](media/microsoft-sign-in-experience-old.jpg)

![Nueva experiencia de inicio de sesión de Microsoft](media/microsoft-sign-in-experience-new.jpg)

## <a name="when-will-the-new-ui-appear"></a>¿Cuándo aparecerá la interfaz de usuario nueva?

A continuación, se muestra una escala de tiempo de cuándo se debe esperar este cambio:

| Período de tiempo | Change |
|-----------|--------|
| Principios de mayo | Comenzará a ver un banner de notificación en la página de inicio de sesión existente. Desde aquí, podrá obtener información acerca de los cambios, así como prepararse para ellos. |
| Mediados de mayo   | El banner de notificación cambiará e incluirá un vínculo para acceder a la versión preliminar de la interfaz de usuario actualizada. En la versión preliminar, aún podrá usar un vínculo para volver a la experiencia existente. |
| Mediados de junio  | La interfaz de usuario de inicio de sesión actualizada estará disponible, así como todos los valores predeterminados para la nueva experiencia de interfaz de usuario. |

## <a name="related-info"></a>Información relacionada

Si es un administrador, podrá encontrar más información acerca de estas actualizaciones en función del producto.

- **Office 365**. Obtenga más información acerca de las actualizaciones en el [Centro de mensajes de Office 365](https://portal.office.com/adminportal/home#/MessageCenter?id=MC133397&MCLinkSource=DigestMail). Debe iniciar sesión con su autenticación de administrador para tener acceso a este sitio.

- **Otros servicios en línea**. Lea esta [entrada de blog](https://cloudblogs.microsoft.com/enterprisemobility/2018/04/04/upcoming-improvements-to-the-azure-ad-sign-in-experience/) para obtener más detalles acerca de las actualizaciones.
